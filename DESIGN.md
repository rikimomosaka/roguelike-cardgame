# DESIGN.md — Roguelike Card Game

Slay the Spire風の協力型マルチプレイ・ローグライクカードゲームのUIデザインガイド。

## ビジュアルコンセプト

- **テーマ**: ダークファンタジー × カードゲーム
- **雰囲気**: 重厚感のあるダンジョン探索。Slay the Spire / Monster Train 系の落ち着いたトーン
- **アートスタイル**: フラットデザインベースに微細なテクスチャとグロウ効果を加えた、モダンなカードゲームUI

## カラーパレット

| 用途 | カラー | 説明 |
|------|--------|------|
| Background (Primary) | `#1a1a2e` | 深い紺 — メイン背景 |
| Background (Secondary) | `#16213e` | やや明るい紺 — パネル背景 |
| Surface | `#0f3460` | カード面・モーダル背景 |
| Accent (Primary) | `#e94560` | 赤 — 攻撃・ダメージ・CTA |

## タイポグラフィ

- **見出し**: `"Cinzel", serif` — ファンタジー感のあるセリフ体
- **本文・UI**: `"Inter", sans-serif` — 可読性重視のサンセリフ
- **数値・ステータス**: `"JetBrains Mono", monospace` — ダメージ数字等

| 要素 | サイズ | ウェイト |
|------|--------|----------|
| h1 (画面タイトル) | 32px | 700 |
| h2 (セクション) | 24px | 600 |
| h3 (カード名) | 18px | 600 |
| body | 14px | 400 |
| caption / label | 12px | 500 |
| ダメージ数字 | 48px | 800 |

## スペーシング・レイアウト

- **基本グリッド**: 4px ベース
- **コンポーネント間隔**: 8px / 12px / 16px / 24px
- **カード間ギャップ**: 12px
- **画面パディング**: 24px
- **ブレークポイント**: 768px (タブレット), 1024px (デスクトップ), 1440px (ワイド)

## 主要画面構成

### 1. タイトル / ロビー画面
- ゲームロゴ（中央配置）
- シングルプレイ / マルチプレイ選択ボタン
- ルーム作成・参加フォーム（合言葉入力）
- 参加者リスト・準備状態表示

### 2. マップ画面
- **扉の選択UI**: 1〜3枚の扉カードを横並びで表示
  - 各扉にイベント種別アイコン（敵・宝箱・ショップ・休憩等）
  - 他プレイヤーの選択状態をリアルタイム表示
- 階層進行のプログレス表示

### 3. バトル画面（メイン）
- **上部**: 敵カードエリア（予告行動アイコン付き）
- **中央**: ユニットエリア（味方ユニット最大3体）
- **下部**: 手札エリア（扇状またはスクロール配置）
- **左**: プレイヤーステータス（HP, エナジー, 防御値）
- **右**: ターゲット選択インジケータ
- **フェーズ表示**: スタート → メイン → バトル の進行をヘッダーに表示

### 4. 報酬画面
- カードパック開封風の演出
- 複数選択肢の並列表示
- ゴールド・アイテム獲得アニメーション

### 5. ショップ画面
- カード・アイテムのグリッド表示
- 所持ゴールド表示
- 購入確認モーダル

## カードデザイン

```
┌──────────────────┐
│  [コスト]    [種別] │  ← エナジーコスト + カード種別アイコン
│                    │
│   [イラスト枠]     │  ← カードアート
│                    │
│──────────────────│
│  カード名          │  ← Cinzel, 16px, bold
│──────────────────│
│  効果テキスト      │  ← Inter, 12px
│  防御+5           │
│                    │
│  [レアリティ帯]    │  ← 下部にレアリティカラーのストライプ
└──────────────────┘
```

- **カードサイズ**: 160px × 224px（手札表示時）
- **レアリティカラー**:
  - Common: `#888888` (グレー)
  - Uncommon: `#48c9b0` (ティール)
  - Rare: `#f0a500` (ゴールド)
  - Legendary: `#e94560` (赤)
- **カード種別**:
  - ユニット: 青枠
  - 武器: オレンジ枠
  - イベント(スキル): 緑枠

## アニメーション・トランジション

| アクション | 演出 | 時間 |
|-----------|------|------|
| カード使用 | 手札からフィールドへスライド + フェードイン | 300ms |
| ダメージ | 数字ポップアップ + 対象シェイク | 400ms |
| 回復 | 緑パーティクル + 数字ポップアップ | 350ms |
| HP変動 | HPバーのスムーズアニメーション | 500ms ease-out |
| フェーズ切替 | 画面中央にフェーズ名テキスト表示 | 800ms |
| カードドロー | 山札から手札へスライドイン | 200ms/枚 |
| 敵撃破 | フェードアウト + パーティクル散乱 | 600ms |
| 扉選択 | 扉が開くアニメーション + 画面遷移 | 1000ms |

## エフェクト・装飾

- **カードホバー**: `scale(1.08)` + `box-shadow` 拡大 + z-index上昇
- **ボタン**: `hover` で微グロウ (`box-shadow: 0 0 12px rgba(accent, 0.4)`)
- **パネル**: `backdrop-filter: blur(8px)` + 半透明背景
- **HPバー**: グラデーション（緑→黄→赤）でHP割合を反映
- **エナジー表示**: パープルのグロウアイコン

## レスポンシブ対応

- **デスクトップ (1024px+)**: フル表示。手札は扇状配置
- **タブレット (768px-1023px)**: カードサイズ縮小。手札はスクロール式
- **モバイル**: 将来対応予定（現時点ではデスクトップ優先）

## アクセシビリティ

- カラーコントラスト比 WCAG AA 準拠（テキスト 4.5:1 以上）
- カード種別はカラー＋アイコンの二重表現（色覚多様性対応）
- キーボード操作対応（Tab でカード選択、Enter で使用）
- ステータス変化は視覚 + テキストで通知
