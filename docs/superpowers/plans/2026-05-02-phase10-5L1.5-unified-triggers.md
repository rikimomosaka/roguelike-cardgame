# Phase 10.5.L1.5 — Relic / Power Trigger 統一 + トリガー網羅追加

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Relic と Power カードの発動タイミング機構を **per-effect Trigger に統一** する。これに合わせて発動タイミングの enum を網羅的に拡張 (relic 専用だった OnPickup / OnBattleStart 等を全 effect で使用可能化)。既存の relic データは effects を空にリセットし、後で再構築する。

**動機:**
- 現状: relic は `RelicDefinition.Trigger` で 1 個固定、power カードは `CardEffect.Trigger` で per-effect。同じ「トリガーで効果発動」概念に 2 つの異なる仕組み存在
- 1 つのレリックの中で複数のタイミング (OnPickup で MaxHP+8、OnBattleStart で 5 ブロック、OnTurnEnd で 1 ドロー など) を同時に持たせたい
- Phase 10.6 (trigger 拡張 + modifier) を控えており、ここで土台を綺麗にしておく

**Architecture:**
- `CardEffect.Trigger` (string?) を **唯一のトリガー指定** とする
- `RelicDefinition.Trigger` フィールドを削除
- `RelicTriggerProcessor.FireInternal` を「def.Trigger == trigger なら全 effects 適用」から「effects ループ内で eff.Trigger == trigger フィルタ」に変更
- `PowerTriggerProcessor` の relic 版バリアント。両者は per-effect Trigger を見るだけなので構造類似 (関数は別のまま)
- 全 relic JSON の effects を `[]` にリセット (ユーザ承諾済)
- DevMeta の `triggers` リストを relic + power 統合形式に拡張、UI も統一

**Tech Stack:** C# .NET 10、xUnit、React 19 + TypeScript、vitest。

**Spec:** `docs/superpowers/specs/2026-05-01-phase10-5-design.md`、本 plan

**スコープ外 (Phase 10.6 で別途実装):**
- run-flow trigger の engine 発火点増設 (OnEnterShop / OnEnterRestSite / OnRest / OnRewardGenerated / OnCardAddedToDeck) — 列挙値だけ追加して dev menu で選択可能化、engine 発火は今回未実装 (作って saved card は engine 実行時 noop)
- modifier-style trigger (BeforeHpLoss / BeforeAttackDamageCalc / BeforeGainBlock / BeforeApplyStatus) — 計算式介入の機構が必要なので 10.6.B
- Passive trigger — 「常時発動」の意味論は modifier 寄り、10.6.B

## Task 1: relic JSON 一括リセット

**Files:**
- Create: `tools/reset-relic-effects.ps1`
- Modify: `src/Core/Data/Relics/*.json` (36 ファイル、スクリプトで)

各 relic の versioned 形式の各 `version.spec` で:
- `effects` を `[]` にリセット
- `trigger` フィールドを削除 (versioned spec 内に存在すれば)
- `description` / `name` / `rarity` / `implemented` 等は保持

PowerShell スクリプト雛形:

```powershell
# tools/reset-relic-effects.ps1
$ErrorActionPreference = 'Stop'
$relicsDir = Join-Path $PSScriptRoot '..\src\Core\Data\Relics'

Get-ChildItem $relicsDir -Filter '*.json' | ForEach-Object {
    $path = $_.FullName
    $utf8 = [System.Text.UTF8Encoding]::new($false)
    $json = [System.IO.File]::ReadAllText($path, $utf8)
    $obj = $json | ConvertFrom-Json
    if ($obj.versions) {
        foreach ($v in $obj.versions) {
            if ($v.spec) {
                if ($v.spec.PSObject.Properties['effects']) { $v.spec.effects = @() }
                if ($v.spec.PSObject.Properties['trigger']) {
                    $v.spec.PSObject.Properties.Remove('trigger')
                }
            }
        }
    }
    $output = $obj | ConvertTo-Json -Depth 20
    [System.IO.File]::WriteAllText($path, $output, $utf8)
    Write-Host "Reset: $($_.Name)"
}
```

確認: `git diff src/Core/Data/Relics/` で 36 ファイル全て effects=[] になっていること。各 relic の description / name / rarity は保持。

### 3.2: 実装

```csharp
private static (BattleState, IReadOnlyList<BattleEvent>) FireInternal(
    BattleState state, string trigger,
    string? deadEnemyInstanceId,
    DataCatalog catalog, IRng rng, int orderStart)
{
    var events = new List<BattleEvent>();
    var s = state;
    int order = orderStart;

    var caster = s.Allies.FirstOrDefault(a => a.DefinitionId == "hero");
    if (caster is null || !caster.IsAlive) return (s, events);

    foreach (var relicId in s.OwnedRelicIds)
    {
        if (!catalog.TryGetRelic(relicId, out var def)) continue;
        if (!def.Implemented) continue;

        // ★ per-effect trigger filtering
        foreach (var eff in def.Effects)
        {
            if (string.IsNullOrEmpty(eff.Trigger)) continue;
            if (eff.Trigger != trigger) continue;

            var (afterEff, evs) = EffectApplier.Apply(s, caster, eff, rng, catalog);
            s = afterEff;
            foreach (var ev in evs)
            {
                var basePrefix = $"relic:{relicId}";
                var suffix = deadEnemyInstanceId is not null ? $";deadEnemy:{deadEnemyInstanceId}" : "";
                events.Add(ev with { Order = order++, Note = (string.IsNullOrEmpty(ev.Note) ? "" : $"{ev.Note};") + basePrefix + suffix });
            }
            caster = s.Allies.FirstOrDefault(a => a.DefinitionId == "hero");
            if (caster is null || !caster.IsAlive) break;
        }
        if (caster is null || !caster.IsAlive) break;
    }

    return (s, events);
}
```
